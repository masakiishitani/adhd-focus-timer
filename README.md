# ADHD Focus Timer

**▶️ [今すぐWebブラウザで使う](https://masakiishitani.github.io/adhd-focus-timer/)**（推奨）

ADHD特化型・強制着手タイマーの「集中継続特化・カウントアップ版」です。カラーリングを紫・ピンク系にカスタマイズした個人利用版です。

> **元ネタ・オリジナル版:** [@tagkaz](https://github.com/tagkaz) さん作の [tagkaz/adhd-focus-timer](https://github.com/tagkaz/adhd-focus-timer) をベースにしています。素晴らしいアプリをありがとうございます！

本プロジェクトは、旧版にあたるポモドーロ用タイマー（究極ポモドーロ）から派生した新規バージョンです。
前作のように「時間が来たら強制的にシステムが休憩時間を挟む」といったポモドーロ手法を廃止し、**「自発的な集中継続の支援」**と**「累積集中時間の可視化」**を最大の目的として再設計されています。

## 目指していること

- **着手のハードルを極限まで下げる:** 「ルーレット開始」等で迷いを断ち切ってすぐに行動に移せます。
- **集中を妨げない:** システム主導の強制終了や休憩遷移を行わず、ユーザーの集中が続く限りタイマーはカウントアップし続けます。
- **「ここまで来た」を実感させる:** 長い集中区間の中で、15分おきに発生する「あえて事務的な」節目通知（フラッシュと文言）により、時間の経過と達成感を可視化します。
- **自己肯定感につながる報酬:** 作業を終了した際に表示される充実した集計画面（本日の累積集中時間、最長連続時間、ベストセッション）が、その日の成果という報酬になります。

---

## 主な特徴

* **カウントアップタイマー:** 00:00から開始し、ユーザーが「一時中断」または「終了」を選ぶまで無限にカウントアップします。
* **15分ごとの節目通知:**
  * 15分, 45分, 75分...（奇数回）：画面が赤く3回フラッシュします。
  * 30分, 60分, 90分...（偶数回）：音声と「〇〇分経過」というトーストメッセージで節目を通知します。
* **状態遷移（一時中断 / 再開 / 終了）:** 離席時には「一時中断」ボタンでタイマーを止められ、中断中の時間は成果（集中時間）には加算されません。
* **集計画面:** 今日の累積集中時間、最長連続時間、着手回数、タスク別ランキングなどの詳細な統計情報を表示します。
* **午前4時区切りの日次集計:** 徹夜などで深夜0時を跨いだ場合でも、午前4時を1日の境界として自動的にセッションを分割・集計します。
* **スマホ完全対応のレスポンシブ設計:** PCだけでなく、スマートフォン等のモバイル端末から開いてもレイアウトが崩れず、快適に操作できるよう最適化されています。
* **環境非依存のデータ保存とエクスポート:** アプリはサーバー不要で `index.html` 単体で動作します。データはブラウザの `localStorage` に保存され、いつでもJSONやCSV形式で手動エクスポート可能です。

## 使い方

### 1. 起動
本プロジェクト内にある `index.html` をお使いのウェブブラウザ（Google Chrome, Safari, Edge など）で開くだけで起動します。ローカルサーバー等は不要です。

### 2. タスクを開始する
- メイン画面の入力欄にこれから取り掛かるタスク名を入力します。
- **「開始 ▶」** ボタンを押すと集中画面へ遷移し、カウントアップが始まります。
- どのタスクから始めるか迷った場合は、**「決められない時（開始）」** ボタン（ルーレット）を押すことでランダムにタスクを決定し、強制的にスタートさせることができます。

### 3. 一時中断と終了の使い分け
集中画面には2つのアクションがあります。
* **「⏸ 一時中断」**: トイレや短い離席時などに使用します。それまでの集中区間が記録され、再開すると新しい区間としてカウントが始まります。一時中断中の経過時間は本日の成果に含まれません。
* **「■ 終了」**: そのタスクを完全に終える、または本日の作業を終える際に使用します。最終的な集中区間が記録され、集計画面が表示されます。

### 4. 本日の成果（集計）を確認する
タスクを終了した直後、またはメイン画面の **「📊 今日の成果」** ボタンから集計画面を開くことができます。
ここにはその日（午前4時区切り）の累積集中時間や、最も長く集中できたセッションの記録などが表示されます。

## データのエクスポート（分析用）

保存された作業記録をスプレッドシートやPython等で分析したい場合、メイン画面または集計画面の **「📥 エクスポート」** (JSON/CSV) ボタンから手動でダウンロードできます。

* **JSON**: 完全なログや設定を含むバックアップ/再集計用のデータ形式。
* **CSV**: 1行1集中区間（タスク名、開始時刻、終了時刻、継続秒数）のExcelやスプレッドシートで扱いやすい形式（BOM付きUTF-8）。

*※ データはすべてユーザーのブラウザ（`localStorage`）内にのみ保存され、外部サーバーへ自動送信されることはありません。*
